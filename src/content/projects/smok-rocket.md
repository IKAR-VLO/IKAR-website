The Suborbitalna Misja demonstracyjna Koła (SMoK) rocket is our core project. High school students designed the physical casing using durable glass-fiber composites.

We calculate motor thrust curves using open-source simulation tools, ensuring custom fuel formulas remain safe and predictable under pressure. Parachutes are set to deploy using barometric microcontrollers at target flight altitudes.
